# B. Calorie Engine

## Step 1 — Calculate BMR

### Mifflin-St Jeor Formula

```text
BMR = 10W + 6.25H - 5A + s
```

Where:

* W = weight (kg)
* H = height (cm)
* A = age
* s = +5 for male / -161 for female

---

## Step 2 — Activity Multiplier

| Activity Level | Multiplier |
| -------------- | ---------- |
| Sedentary      | 1.2        |
| Light          | 1.375      |
| Moderate       | 1.55       |
| Heavy          | 1.725      |
| Athlete        | 1.9        |

---

## Step 3 — Calculate TDEE

```text
TDEE = BMR × ActivityMultiplier
```

---

## Step 4 — Goal Calories

### Fat Loss

```text
GoalCalories = TDEE - 300~700
```

### Muscle Gain

```text
GoalCalories = TDEE + 200~500
```

### Maintain

```text
GoalCalories = TDEE
```

# 3. Weight Prediction System

## Core Rule

```text
7700 kcal ≈ 1kg fat
```

---

## Prediction Formula

```text
PredictedWeightChange = CalorieDelta / 7700
```

---

## Example

500 kcal deficit/day:

```text
≈ 0.45kg loss/week
```

---

## Timeline Prediction

Example:

```text
Current Weight: 80kg
Target Weight: 70kg
Need to lose: 10kg
```

Total deficit needed:

```text
10 × 7700 = 77000 kcal
```

If:

```text
500 kcal deficit/day
```

Then:

```text
≈ 154 days
```

# 4. Adaptive Metabolism Engine

## Why Important

Human metabolism changes over time.

As users:

* lose weight
* gain muscle
* diet longer

their TDEE changes.

---

# Weekly Adaptive Calculation

Every 7 days:

Collect:

* Average calories consumed
* Weight trend
* Steps
* Exercise calories

Estimate actual TDEE:

```text
ActualTDEE = AvgCalories - (7700 × WeeklyWeightChange / 7)
```

---

## Example

User:

```text
eats 2000 kcal/day
```

Weight trend:

```text
loses 0.5kg/week
```

Estimated deficit:

```text
≈ 550 kcal/day
```

Estimated TDEE:

```text
≈ 2550 kcal/day
```
